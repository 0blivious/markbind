MarkBind

MarkBind is a tool built on top of Markdown syntax, for creating websites that can go beyond a collection of linear web documents. Markdown is particularly suitable for creating course Websites and online instruction manuals. Examples of websites built using Markbind: [Example 1][Example 2]
